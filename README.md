# calcul_de_l-imc_

## Français

Ce projet a pour objectif de me permettre de m'améliorer en Python.  
Il consiste en la création d'un script simple pour calculer l'IMC (Indice de Masse Corporelle).

### Utilisation

Clonez le dépôt puis lancez le script principal :

```bash
python main.py
```

## English

This project aims to help me improve my Python skills.

### Usage

```bash
python main.py
```
